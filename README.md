# Watcher


## Getting Started
```bash
$ git clone https://github.com/dapinto8/watcher.git
$ cd watcher
$ npm install
$ npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

#### Environment
```bash
SITE_URL=
API_URL=
API_KEY=
IMAGES_URL=
```

## Build and serve
```bash
$ npm run build 
$ npm run start
```

## Runing tests
#### Run all tests
```bash
$ npm run test
```
#### Run specific test
```bash
$ npm test -- /path/to/test/file
```
